## Quick run

**install flask**
sudo apt install python3-flask

**run flask app**
flask --app hello run

## Demo

**install bundler**
pip install pipenv
apt install pipenv

**use pipenv to create a Python 3 (--three) virtualenv for our project**
pipenv --three

**install flask a dependency on our project**
pipenv install flask

**create the file**
touch bootstrap.sh

**make it executable**
chmod +x bootstrap.sh
